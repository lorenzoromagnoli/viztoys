**Keyboard Shortcuts:**
- `B` - Cycle through modes (Generate â†’ Brush â†’ Spawn)
- `H` - Hide/show instructions
- `P` - Play/pause particle simulation
- `S` - Export as SVG

- **Generate Mode**: Mouse controls attractor
